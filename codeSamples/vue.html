What You Will Be Doing

Build beautiful UI components and pages that will be used by thousands of people of all ages and help people understand their genomics and test results.


Work alongside other developers and UI/UX designers to build business logic and features to add to the existing Frontend application. Build a generic and re-usable front end architecture that can be easily customized per client while taking advantage of our platform APIs. Develop E-commerce components using Shopify and also build both desktop and mobile friendly version of the pages. Investigate other new technologies to decrease our time to deployment.


Develop using the latest modern frameworks such as VueJs and Angular and use Sass, Bootstrap and other CSS libraries to create beautiful and user friendly designs. 


What You Must have for this Position

  • Must have 2 years VueJS experiance
  • Experience coder using a modern JS Framework (VueJS, Angular, or React)
  • Always be thinking responsiveness and multi browser compatibility
  • Proficient and quick with SASS or LESS or similar
  • Experience with Bootstrap or other UI frameworks
  • Must love HTML, CSS, and beautiful UIs


What is nice to have for this Position

  • Experience with CSS animation is a plus
  • Experience working with 3rd party APIs

About ixlayer

ixLayer is an end-to-end health technology platform and cloud. The platform integrates to genetic labs, physician networks, and on demand genetic counseling; through a HIPAA compliant platform.


Ixlayer is currently powering nationwide COVID-19 Testing programs at scale including government contracts on a federal level. 


ixLayer technology SaaS platform provides health systems and individual physicians with the tools they need to order testing on patients. The technology platform and cloud platform enables companies such as Health organizations, Clinical labs, Pharmaceuticals and Consumer brands to offer clinical diagnostic programs.
